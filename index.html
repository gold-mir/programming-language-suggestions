You should learn C#

C# is a powerful back-end language, originally developed by Microsoft, that's extremely popular among larger companies. If you want to work on internal software, you should probably learn C#. Additionally, Unity, one of the most popular cross-platform 3D game engines uses C#, making it an ideal language for people looking to go into game development.

You should learn Java

Java's been around for a long time, and as a result a lot of companies use it. Java is relatively unique among compiled languages in the fact that it is compiled into "Java Bytecode" instead of directly into an executable file, allowing the same Java code to run any device with a Java interpreter. Java is also the language most Android apps are developed in, making it by far the best choice for anyone primarily interested in developing android apps.

You should learn Ruby

Ruby is an interpreted language that's known for it's readability and ease of use. Ruby is primarily known for the Ruby on Rails web framework, which allows a developer to create complex web applications extremely quickly by use of an extensive set of helper libraries. Ruby style is heavily dictated by convention, and Ruby developers often emphasize doing things "the Ruby way", as the language has a number of ease of use features that are unlike most other programming languages.
